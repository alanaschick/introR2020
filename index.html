About the Workshop

This is a two day intensive workshop to teach scientists (students, postdocs, PIs, and staff) in the biological and medical sciences to use R for typical data analysis tasks they might encounter routinely. No prior knowledge of R is expected and workshop attendees can expect to come away with a skill set that is immediately translatable to their respective data tasks. .

A rough overview of the topics covered is as follows:

  • How to install and update R
  • How to use the RStudio IDE
  • How to import, subset, and manipulate data in R
  • R's basic data structures: vectors, matrices, lists, data frames
  • How to make publication-quality plots using the ggplot2 package
  • How to understand some common R errors and deal with them
  • How to find and evaluate R packages for a particular analysis
